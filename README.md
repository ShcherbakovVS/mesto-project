**Интро**

Социальная сеть "Mesto" про путешествия. В ней вы можете создать профиль, выкладывать в нем свои фото, ставить лайки.
Это учебный проект сложнее предыдущих, в нем используются модальные окна, формы, кнопки, интерактивность. Проект создан при помощи html, css, js

Доставать картинки предстоит из Фигмы. Это расхожая практика, поэтому полезно потренироваться.
